Class Description
Request The Request class provides Controller and Service objects with access to form and query parameters, cookies, the web session and the web server environment. It also makes the app language available, and the requested content type and HTTP method.
Request_cookie The Request_cookie class enables the app to get and set HTTP cookies
Request_server The Request_server class enables the app to get (and set!) server settings
Request_session The Request_session class enables the app to get and set session variables
