YAWF

Description
Description | Descendents | Methods (details)

The YAWF object responds to web requests by creating an "App" object, a "Controller" object, then rendering a web response.

The YAWF class should be inherited by all classes so that it may handle exceptions and unknown method calls. It provides useful static methods such as "hook" to hook unknown method calls, and "prop" to register props for mock tests or other kinds of dependency injection.

The safe way to finish responding to a web request is to call the YAWF::finish() method because this will perform logging and benchmarking, unlike a simple "exit" call.
