static login (line 81)

Require the web user to login using basic auth.

Either pass the required username and password, or pass an object in place of the username, and have the object provide a "basic_auth()" method with username and password parameters, returning TRUE if the username and password are authorized.

Note that this method will call YAWF::finish() by default, to cause the web request to finish.

  • return: whether the login succeeded
  • access: public
Boolean login (String/Object $username, [String $password = ''], [Boolean $do_return = FALSE])
  • String/Object $username: the username or object with "basic_auth()"
  • String $password: the required password (when username is a string)
  • Boolean $do_return: whether to always return (default is FALSE)
