HTML

Description
Description | Vars (details) | Methods (details)

This helper makes it easy to insert HTML into views in a clean consistent manner. The form methods support object properties:

  1.  $html = HTML::form_open('form-id');
  2.  $html .= HTML::input('user->name');
  3.  $html .= HTML::submit('OK');
  4.  $html .= HTML::form_close();

  • author: Kevin Hutchinson <kevin@guanoo.com>
