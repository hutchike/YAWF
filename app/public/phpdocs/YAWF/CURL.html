CURL

Description
Description | Methods (details)

Wrap the PHP cURL extension with handy static function calls for the HTTP methods "get", "delete", "post" and "put". Note that the URL in function calls may be prefixed using a basic authentication login like "https://user:pass@site.com/a/12".

  • author: Kevin Hutchinson <kevin@guanoo.com>
