Queue

Description
Description | Methods (details) Constants (details)

Provide a simple helper to write objects to a queue and read them from the queue (either by blocking - default behavior - or non-blocking).

The objects written and read from the queue retain their PHP object classes (provided classes are already loaded into the program). The objects are written as files in queue folders.

  • author: Kevin Hutchinson <kevin@guanoo.com>
