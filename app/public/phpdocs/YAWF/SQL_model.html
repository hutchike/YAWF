SQL_model

Description
Description | Descendents | Vars | Methods (details)

Implements interfaces:

  • Modelled
  • Persisted
  • Validated

The SQL_model class links data objects to storage engines that support SQL queries such as MySQL, SQLite2 & SQLite3.

  • author: Kevin Hutchinson <kevin@guanoo.com>
