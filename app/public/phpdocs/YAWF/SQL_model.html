setup (line 57)

Setup a model by calling methods such as "set_id_field", "set_virtual" and "set_timestamp". This method should be overriden in your subclass:

  1.  // Subclass in your models like this:
  2.  
  3.  public function setup()
  4.  {
  5.      $this->set_id_field('table_id_field');
  6.      $this->set_virtual('transient_field');
  7.      $this->set_timestamp('created_at', 'updated_at');
  8.      $this->validates('email', 'is_valid_email'); // when a "Model" object
  9.  }
