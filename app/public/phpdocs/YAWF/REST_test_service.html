REST_test_service

Description
Description | Vars | Methods (details)

The REST_test_service class is used to test the REST controller.

Basically this class simply returns the data parameters that are passed to each of its REST methods, so they may be tested easily. These methods are each called by the REST_test_controller class. To run the tests, visit an address like http://localhost/REST_test
