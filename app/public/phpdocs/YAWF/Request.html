Request

Description | Descendents | Vars (details) | Methods (details)

Implements interfaces:

  • Mailer

The Request class provides Controller and Service objects with access to form and query parameters, cookies, the web session and the web server environment. It also makes the app language available, and the requested content type and HTTP method.

  • author: Kevin Hutchinson <kevin@guanoo.com>
