Translate

Description
Description | Methods (details)

Provide useful text translation methods after loading the YAML

"validate" and "translate" config files from the app "configs" folder. The methods provided are "add_translations" to add new translations overriding current ones, "into" to translate some text and "validate" to check that translations aren't missing.
