Remote

Description
Description | Descendents | Vars | Methods (details) Constants (details)

Implements interfaces:

  • Modelled
  • Persisted
  • Validated

The Remote class provides remote data manipulation capabilities via REST by providing a facade to a data model object via the Modelled, Persisted and Validated interfaces.

  • author: Kevin Hutchinson <kevin@guanoo.com>
