Relating_model

Description
Description | Descendents | Vars | Methods (details) Constants (details)

Implements interfaces:

  • Modelled
  • Persisted
  • Validated

The Relating_model adds relationships to models via methods "belongs_to", "has_a" and "has_many". Unlike other "sexier" frameworks, this class doesn't support many-to-many relations.

  • author: Kevin Hutchinson <kevin@guanoo.com>
