GeoIP

Description
Description | Methods (details) Constants (details)

This helper wraps the "Net/GeoIP" plugin from MaxMind with a simple object-oriented interface to provide the IP address, the country name and code, and the location (i.e. city).

For example, if you're using the GeoIPCity.dat data file:

  1.  load_tool('GeoIP');
  2.  $geo = new GeoIP('app/data/GeoIPCity.dat');
  3.  $country = $geo->country_name();
  4.  $code = $geo->country_code();
  5.  $city = $geo->city();
  6.  $rgn = $geo->region();
  7.  $zip = $geo->zip_code();
  8.  $dma = $geo->dma_code();
  9.  $lat = $geo->latitude();
  10.  $long = $geo->longitude();

...or to use the default GeoIP.dat file in /usr/share/GeoIP:

  1.  load_tool('GeoIP');
  2.  $geo = new GeoIP('app/data/GeoIPCity.dat');
  3.  $country = $geo->country_name();
  4.  $code = $geo->country_code();
  5.  // Can't use the city data methods
  6.  // coz they will just return NULL
