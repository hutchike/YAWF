Daemon

Description
Description | Vars | Methods (details)

The Daemon class is a type of command that is designed to run

in the background without any duplicate processes. It achieves this by keeping note of its PID in a pidfile in the "/tmp/pids" directory. A Daemon object can't be created if another daemon is running simultaneously.
