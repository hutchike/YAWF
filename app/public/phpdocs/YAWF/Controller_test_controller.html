Controller_test_controller

Description
Description | Vars | Methods (details) Constants (details)

The Controller_test_controller class tests the "Controller" and "App" classes in the YAWF "lib" directory by checking that they provide the services expected, e.g. flashes, cookies, sessions, params, redirects.

Basically, it's a controller to test controllers. All a bit recursive.

  • author: Kevin Hutchinson <kevin@guanoo.com>
