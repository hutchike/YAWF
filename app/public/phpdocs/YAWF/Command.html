Command

Description
Description | Descendents | Vars (details) | Methods (details)

The Command class provides a base class to write commands to

run from the command line. Commands have a path, a name, some options (optionally) and some arguments (optionally). Options take the form "-opt1=a -opt2=b" on the command line, whereas arguments are simply listed at the end like "arg1 arg2 arg3".

Command logging is written to a ".command" log, or to the ".command.test" log when we're running command tests, for example when the "-test" option is passed to the command.
